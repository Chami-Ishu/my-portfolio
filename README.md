## 🎨 Features

- ⭐ Animated star particle background
- ⌨️ Typewriter effect with multiple roles
- 📊 Animated skill progress bars
- 🎞️ Scroll reveal animations
- 💙 Blue & Cyan color theme
- 📱 Fully responsive design
- 🌙 Dark mode UI
